# 待实现：账表与结账

## 1. 账簿查询

以当前账套的已过账凭证为基础，按科目维度汇总余额和发生额，提供标准会计账簿视图。

### 1.1 总分类账

展示每个科目的期初余额、本期借方发生额、本期贷方发生额、期末余额。支持按会计期间、科目范围筛选。

- **取数来源**：科目的期初余额 + 当前期间已过账凭证的借/贷发生额
- **联查**：双击某行可穿透到该科目的明细分类账

### 1.2 明细分类账

展示某个科目在指定期间内所有已过账凭证的逐笔记录，含凭证号、摘要、借方金额、贷方金额、余额。

- **筛选**：会计期间、科目代码
- **联查**：双击某笔可打开该凭证的只读详情

### 1.3 科目余额表

按科目层级展示每个科目的期初余额、本期借方、本期贷方、期末余额。父级科目余额自动汇总子级。

- **期间选择**：可切换不同会计期间
- **余额方向**：根据科目的余额方向属性，正常余额以正数显示，异常余额标红

### 1.4 试算平衡表

展示所有科目的期初余额、本期发生额、期末余额，校验借贷平衡。

- **校验规则**：期初借=期初贷、本期借=本期贷、期末借=期末贷
- **不平衡时**：标红提示，方便定位问题

### 1.5 多栏账

对某一科目按辅助核算维度（如部门、客户）分多列展开，每列显示该维度的发生额和余额。

## 2. 财务报表

### 2.1 资产负债表

按固定模板（项目行 + 期末余额 + 年初余额）展示资产、负债、所有者权益三大类。

- **取数逻辑**：每个报表项目对应一个取数公式，公式从科目余额表的期末余额取数
- **校验**：资产合计 = 负债合计 + 所有者权益合计

### 2.2 利润表

按固定模板（项目行 + 本月数 + 本年累计数）展示收入、成本、费用、利润。

- **取数逻辑**：从损益类科目的本期发生额取数
- **期间**：可选择月份，年中各期显示本期数据，年末显示全年

### 2.3 现金流量表

按经营活动、投资活动、筹资活动三大类展示现金流入流出。

- **取数方式**：指定现金类科目，从对应的凭证分录汇总
- **初期可简化**：人工指定现金流量项目，不要求凭证级自动分类

## 3. 期末结账

### 3.1 损益结转

将损益类科目（收入、成本、费用等）的本期发生额结转到"本年利润"科目，生成结转凭证。

- **触发**：用户选择期末期间，系统自动生成一张或多张结转凭证
- **规则**：收入类科目贷方余额 → 借方结转；费用/成本类科目借方余额 → 贷方结转；差额进"本年利润"
- **约束**：结转前需保证本期所有凭证已过账；已结转的期间可再次结转（覆盖上次结转凭证）

### 3.2 期末关账

将指定期间标记为"已关闭"，关闭后禁止对该期间新增、修改、删除凭证。

- **前置条件**：本期凭证全部过账 + 损益结转已完成
- **效果**：在凭证录入/修改时校验期间状态，已关账则拒绝操作
- **反关账**：admin 权限可执行，记录操作日志，用于月末调整

## 4. 科目属性扩展（前置依赖）

在实现账表和结账之前，需要先扩展科目表的部分字段：

- **余额方向**：借方 / 贷方 — 用于科目余额表正常余额判断
- **是否辅助核算**：标记该科目是否需要辅助核算维度（此字段为后续辅助核算功能预留）
- **是否现金科目**：标记现金/银行类科目 — 用于现金流量表取数
