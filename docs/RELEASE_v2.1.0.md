# OpenFina v2.1.0 Release Notes

发布日期：2026-05-09

## 发布定位

OpenFina v2.1.0 是一次权限系统升级。本版本引入基于角色的访问控制（RBAC），将用户划分为系统管理员（admin）和财务主管（manager）两种角色，在 UI 层和模型层实现双层权限校验。同时新增命令行调试工具，便于开发和排障。

本版本兼容 v2.0.0 的用户数据，旧用户库会自动迁移添加角色相关列。

## 新增功能

### 角色与权限体系

- 新增两种角色：系统管理员（admin）和财务主管（manager）
- 角色权限分离：
  - **财务主管**：凭证录入/编辑/删除、凭证审核/取消审核、凭证过账/取消过账、凭证查询、凭证汇总、期末结账、添加下级科目
  - **系统管理员**：凭证查询（只读）、凭证汇总、添加/删除一级科目、账套创建/删除、用户管理、批量取消过账、批量取消审核、查看系统日志
  - **核心规则**：管理员拥有系统级权限但零业务操作权限（权责分离），财务主管不能审核/过账自己的凭证（沿用已有规则）
- 权限校验在 UI 层（按钮可见性）和模型层（操作守卫）双重生效

### 默认管理员引导

- 首次启动时，若系统中无管理员账号，自动创建默认管理员 `admin` / `admin123`
- 默认管理员首次登录时强制修改密码，关闭按钮被禁用，无法绕过

### 用户管理

- 管理员可在左侧导航栏点击"用户管理"打开弹窗
- 支持创建用户（可选择角色）、启用/禁用账号、切换角色、重置密码
- 重置密码后用户下次登录需修改密码
- 保护机制：不能移除最后一个管理员、不能修改自己的角色或状态

### 命令行调试工具

- 新增 `cli.py` 统一命令行入口，`argparse` 子命令风格
- 四个命令组：
  - `user` — 用户列表、详情、创建、启用/禁用、角色切换、密码重置
  - `bookset` — 账套列表、创建、设置默认
  - `voucher` — 凭证查询、单张查看、科目汇总、批量取消过账/审核
  - `subject` — 科目列表、树形展示、添加下级、删除
- 支持文本表格（默认）和 JSON（`--json`）两种输出格式

### 切换用户

- 主界面标题栏新增"切换用户"按钮，所有角色可见
- 点击后清除自动登录设置，返回登录界面

## 数据库变更

- `users` 表新增两列：
  - `role TEXT NOT NULL DEFAULT 'manager'`
  - `must_change_password INTEGER NOT NULL DEFAULT 0`
- 旧用户库启动时自动迁移，不破坏现有数据

## 新增文件

| 文件 | 说明 |
|------|------|
| `ui/password_change_dialog.py` | 密码修改对话框（含强制修改模式） |
| `ui/user_management.py` | 用户管理弹窗和创建用户对话框 |
| `cli.py` | 命令行调试工具 |
| `tests/test_admin_methods.py` | 管理员方法单元测试（14 个用例） |

## 主要修改文件

| 文件 | 变更 |
|------|------|
| `models/bookset.py` | users 表迁移、认证方法返回角色、新增管理员方法 |
| `models/voucher.py` | 凭证操作权限守卫、批量取消过账/审核 |
| `app/bootstrap.py` | 默认管理员引导 |
| `app/main.py` | 角色 UI 可见性、用户管理入口、切换用户按钮 |
| `ui/login.py` | 登录流程强制密码修改检查 |

## 已知限制

- 期末结账、逆向结账、系统日志查看等功能仍在规划中
- 凭证操作（录入/审核/过账）的 CLI 接口因参数复杂暂未暴露
- 账簿、完整报表、现金流量等模块仍处于预留阶段

## 推荐测试路径

1. 首次启动 → 自动创建默认管理员 `admin`
2. 以 `admin`/`admin123` 登录 → 强制修改密码 → 进入主界面
3. 管理员界面验证：左侧有"用户管理"入口，凭证处理可见但录入/审核/过账按钮已隐藏
4. 打开用户管理 → 创建财务主管用户 → 切换到财务主管登录
5. 财务主管界面验证：可录入、审核、过账凭证，无用户管理入口
6. 财务主管尝试审核自己制作的凭证 → 模型层拒绝
7. 管理员尝试通过 CLI 直接调用凭证录入 → 权限检查生效
8. 使用 `python cli.py user list` 查看用户列表
9. 使用 `python cli.py voucher list --db data/booksets/xxx.db --start 2026-01-01` 查询凭证
10. 点击标题栏"切换用户" → 返回登录界面
